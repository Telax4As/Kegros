18:04


Данный проект представляет собой сайт с полезной и порой редко встречаемой информациях об университетах Казахстана.

Основным сайтом является index.html, остальные сайты с разширением .html являются сайтами с ифнормацией об университетах. Все стили всех сайтов находятся в style.css.
